Dashlane to 1Password

What does this do?

This is a tool that converts the Dashlane exported file (Unsecured archive (readable) in CSV format), to an 1Password compatible file. I hope this helps people who are switching from Dashlane to 1Password. If you experience an issue, please open an issue on the GitHub project page.

Is it secure? We're talking about my passwords here!

To be clear, your file will !! never ever !! touch a server, this is all being done on the client side in your browser. You can look at the code on: https://github.com/vblinden/dashlane-to-1password.

What do I do?

Upload your Dashlane exported file (CSV format) with the file uploader to your right, and it will automatically generate and download the 1Password compatible file.
